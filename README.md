# Longest Increasing Subarray

Bu proqram verilmiş massivdə ardıcıl yerləşən ən uzun artan elementlər ardıcıllığının uzunluğunu tapır.

## Alqoritm
Massiv üzərindən bir dəfə keçilir. Əgər cari element əvvəlkindən böyükdürsə, ardıcıllıq davam edir, əks halda sıfırlanır.

## Nümunə

Input:
1 2 3 2 3 4 5 1

Output:
4
